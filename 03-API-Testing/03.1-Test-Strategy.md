# API Test Strategy

## Scope
Covers manual and automated testing of HCMS APIs including case management, import, and export.

---

## Testing Types

### Manual Testing
Used for:
- Exploratory testing
- New feature validation
- Edge case verification
- One-time validation scenarios

### Automated Testing
Used for:
- Regression testing
- Repeated API validation
- Stable endpoints (Create, Get, Update, Delete, Export)

---

## Automation Scope
- Create Case
- Get Case
- Update Case
- Delete Case
- Export Case

---

## Manual Scope
- Import Case (file variability)
- Negative scenarios exploration
- Boundary testing
- Unusual data combinations

---

## Tools
- Postman (manual API testing)
- Newman / Robot Framework (automation execution)
