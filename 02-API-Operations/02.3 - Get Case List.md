# Get Case List

## Purpose

Retrieve all healthcare cases.

---

## Endpoint

GET /api/cases

---

## Request Headers

| Header | Value |
|---------|-------|
| Authorization | Bearer Token |

---

## Success Response

Status Code

200 OK

```json
[
  {
    "caseId": "HCMS-10001",
    "patientName": "John Doe",
    "status": "Open"
  },
  {
    "caseId": "HCMS-10002",
    "patientName": "Jane Smith",
    "status": "Closed"
  }
]
```

---

## Common Error Responses

| Status | Description |
|---------|-------------|
| 401 | Unauthorized |
| 500 | Internal server error |
