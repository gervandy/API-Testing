# Get Case List

## Business Purpose
Retrieve a list of all healthcare cases.

---

## Endpoint
GET /api/cases

---

## Headers
| Header | Value |
|--------|------|
| Authorization | Bearer Token |

---

## Success Response
200 OK

```json
[
  {
    "caseId": "HCMS-10001",
    "patientName": "John Doe",
    "status": "Open"
  },
  {
    "caseId": "HCMS-10002",
    "patientName": "Jane Smith",
    "status": "Closed"
  }
]
```

---

## Error Responses
- 401 Unauthorized
- 500 Server error

---

## Validation Points
- Status code is correct
- Response is a valid array
- Required fields exist (caseId, patientName, status)
- Data structure is consistent

---

## Test Scenarios

### Positive
- Retrieve all cases successfully

### Negative
- Unauthorized request
- Server failure response

### Boundary
- Empty case list
- Large dataset response
