# Create Case

## Purpose

Create a new healthcare case.

---

## Endpoint

POST /api/cases

---

## Request Headers

| Header | Value |
|---------|-------|
| Authorization | Bearer Token |
| Content-Type | application/json |

---

## Request Body

```json
{
  "patientId": "P10001",
  "patientName": "John Doe",
  "medication": "Aspirin",
  "reaction": "Skin Rash",
  "status": "Open"
}
```

---

## Success Response

Status Code

201 Created

```json
{
  "caseId": "HCMS-10001",
  "message": "Case created successfully"
}
```

---

## Common Error Responses

| Status | Description |
|---------|-------------|
| 400 | Validation failed |
| 401 | Unauthorized |
| 409 | Duplicate case |
| 500 | Internal server error |
