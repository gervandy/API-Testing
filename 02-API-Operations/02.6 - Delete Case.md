# Delete Case

## Business Purpose
Delete an existing healthcare case by case ID.

---

## Endpoint
DELETE /api/cases/{caseId}

---

## Path Parameters
| Parameter | Type |
|----------|------|
| caseId | String |

Example: HCMS-10001

---

## Success Response
204 No Content

(No response body)

---

## Error Responses
- 401 Unauthorized
- 404 Case not found
- 500 Server error

---

## Validation Points
- Status code is 204
- Case is no longer retrievable
- Deleted case returns 404 on GET request
- No residual data remains in system

---

## Test Scenarios

### Positive
- Delete valid case successfully

### Negative
- Delete non-existent case
- Unauthorized delete request

### Boundary
- Delete already deleted case
- Delete with invalid case ID format
