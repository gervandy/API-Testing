# Authentication

## Purpose

Authenticate a user and obtain an access token for subsequent API requests.

---

## Endpoint

POST /api/auth/login

---

## Request Headers

| Header | Value |
|---------|-------|
| Content-Type | application/json |

---

## Request Body

```json
{
  "username": "qa.user",
  "password": "Password123"
}
```

---

## Success Response

Status Code

200 OK

Example Response

```json
{
  "accessToken": "eyJhbGciOi...",
  "expiresIn": 3600
}
```

---

## Common Error Responses

| Status | Description |
|---------|-------------|
| 400 | Invalid request |
| 401 | Invalid username or password |
| 500 | Internal server error |
