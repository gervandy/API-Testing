# System Overview

## Healthcare Case Management System (HCMS)

The Healthcare Case Management System (HCMS) stores and manages healthcare cases submitted by medical professionals.

Each healthcare case may contain:

- Patient Information
- Medication Details
- Adverse Reactions
- Important Dates
- Case Status
- Supporting Documents

---

## API Purpose

The HCMS APIs allow external applications and internal services to manage healthcare cases without interacting directly with the user interface.

The APIs support common business operations including:

- Create Case
- Retrieve Case
- Update Case
- Delete Case
- Import Cases
- Export Cases

---

## Testing Objective

API testing validates that backend services correctly process requests, enforce business rules, and return expected responses.
